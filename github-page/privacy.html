Privacy Policy

Effective date: April 27, 2026

Facts&Quiz respects your privacy.

Data Collection

This extension does not collect, sell, share, or transmit any personal information.

Local Storage

The extension may store user preferences, settings, quiz progress, and favorite facts locally in the browser using Chrome storage. This data stays on the user's device and is not sent to any external server.

Internet Access

This extension does not require an account and does not send user data to third-party services.

Permissions

The extension uses the storage permission only to save local settings, favorites, progress, and quiz preferences.

Third-Party Services

This extension does not use analytics, advertising networks, tracking tools, or third-party data services.

Changes to This Policy

This Privacy Policy may be updated if the extension's functionality changes. Any future changes will be reflected on this page.

Contact

If you have any questions about this Privacy Policy, you can contact the developer through the project's GitHub repository.
