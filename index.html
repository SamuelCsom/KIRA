How to install

This extension currently uses manual installation through Chrome Developer Mode.

  1. Download the project ZIP using the button above and extract it.
  2. Open Chrome and go to chrome://extensions.
  3. Enable Developer mode in the top-right corner.
  4. Click Load unpacked and select the extracted folder.
  5. Pin FAQ to the toolbar and start browsing facts and quizzes.
